Running NounChunkerGenerator Will output .pos tag and .tchunk files by reading outputs from
Brandies Chinese System. The format should be consistent to .out.txt file generated from Brandies
System

Instructions for running NounChunkerGenerator:
javac NounChunkerGenerator.java
java NounChunkerGenerator filename background
or 
java NounChunkerGenerator filename foreground
to specify the type of the file you are dealing with
