Software Heritage

 est un projet initié par Roberto Di Cosmo et Stefano Zacchiroli en 2014. L'ambition est d'archiver tout le code source créé par l'humanité comme Internet Archive ambitionne d'archiver toutes les pages internet. L'ouverture de l'archive se fait en 2016 et est soutenue par l'Inria. Début 2017, l'UNESCO soutient le projet.

Le lancement officiel est le . À cette date, près de 4.5 milliards de fichiers sources pour 83 millions de projets sont archivés. On y retrouve tous les codes sources disponibles sur GitHub, l'histoire du développement Unix ainsi que le code source de la mission Apollo 11.

Le 25 septembre 2018, les dépôts de logiciels sur HAL sont connectés à Software Heritage.
